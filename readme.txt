== Description ==
Cette astuce permet d\'améliorer la navigation de votre site en redirigeant vers le seul article d\'une recherche fructueuse ou d\'un tag. Au lieu de rediriger l’utilisateur vers la page de résultats de recherche et le laisser cliquer sur le lien de l’article, on va directement le rediriger vers l’article correspondant à sa recherche. Ce plugin s\'appuie sur une astuce soumise par Jean-David DAVIET (http://goo.gl/TpXl0G).

== Installation ==
To install this plugin :
- download the zipped file, on https://github.com/yvangodard/WP-redirect-to-post-if-only-one-result/archive/master.zip),
- upload it on WordPress, through the \"Plugins\" menu in WordPress,
- and activate it.
